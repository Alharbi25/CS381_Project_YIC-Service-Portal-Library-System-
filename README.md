# Library Management System 

## Overview
A web-based library management system built with **PHP (PDO)** and **MySQL**.

## Features
- **Role-based Login:** Admin and Student roles.
- **Library Catalog:** Browse and search for books.
- **Borrowing System:** Students can borrow and return books.
- **Admin Panel:** Manage books and track student loans.
- **Security:** Uses password hashing and prepared statements.

## Setup
1. Import `database.sql` into phpMyAdmin.
2. Place files in XAMPP `htdocs` folder.
3. Login as Admin: `admin@yic.edu.sa` | Pass: `123456`.
